instrumentation layers

Entropy Capsule Engine
entropy_capsule.py
Instruments LLM output entropy per action. Tracks token entropy, confidence collapse, hallucination, coherence drift, and instability spikes. The agent-side mirror of external benchmarking.
Session Drift Monitor
session_drift_monitor.py
Behavioral integrity monitoring with HALT logic. Signals semantic drift, scope creep, noise violations, and parse failures. Blocks anomalous action streams before dispatch.
Payload Mutator
payload_mutator.py
Feedback-driven personality selection across six profiles. Five mutation operators. Fitness scores update per channel attempt — the framework remembers what failed.
mTLS Mesh
mtls_mesh.py
NaCl Box peer mesh with HMAC-SHA256 handshake tokens keyed on private key material. Quarantine state for peers that fail cryptographic verification.
Session Report
tools/session_report.py
Unified post-session artifact. Pulls drift, entropy, and mutation state into one timeline. Fires automatically on console exit. JSON export for offline analysis.
Session Comparator
tools/compare_sessions.py
Diffs two session JSON reports. Surfaces score deltas, new and resolved signals, fitness shifts, and timeline entropy correlation between runs.

security hardening

✓ HMAC-SHA256 handshake
Mesh tokens keyed on private key material. Not forgeable from public data. Timing-safe comparison via hmac.compare_digest().
✓ HKDF plugin keys
AES-GCM keys derived via HKDF-SHA256 with ZDS_PLUGIN_SALT. Not recoverable from agent ID alone.
✓ Ed25519 plugin signing
exec() gated on Ed25519 signature verification. Unsigned plugins rejected when ZDS_PLUGIN_PUBKEY is configured.
✓ TLS cert pinning
FingerprintAdapter pins C2 cert by SHA256 fingerprint. Falls back to system CA. Never verify=False.
